# Raven Theme for Brackets

Dark color scheme based on Dreamweaver CC's Raven theme. Now with better preprocessor support.

## Installation
* Open Brackets
* Open the Extension Manager
* Switch to "Themes" tab
* Search for "Raven"
* Click "Install"
